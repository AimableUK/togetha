# Togetha

Togetha is a collaborative workspace application designed for real-time
idea sharing, note-taking, and visual collaboration. It combines
features inspired by tools like Notion and FigJam into a clean, minimal
interface.

## Features

-   Real-time collaboration
-   Rich text editor
-   Canvas for brainstorming and diagrams
-   Cross-device synchronization
-   Offline support
-   Clean and minimal UI

## Tech Stack

-   Frontend: React (Vite)
-   State Management: Zustand / SWR
-   Styling: Tailwind CSS
-   Deployment: Vercel

## Vision

Togetha aims to become a lightweight, powerful productivity tool for
teams and individuals to organize thoughts, collaborate seamlessly, and
visualize ideas in one place.

## Future Improvements

-   Authentication system
-   Role-based collaboration
-   Backend integration (Node.js / Django)
-   Real-time database (Firebase / Supabase / Redis)
-   File uploads and media support
-   Advanced canvas tools

## Getting Started

1.  Clone the repository

2.  Install dependencies:

    ``` bash
    npm install
    ```

3.  Run the development server:

    ``` bash
    npm run dev
    ```

## Deployment

Deploy easily using Vercel: - Connect your repository - Set environment
variables if needed - Deploy
